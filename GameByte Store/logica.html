Lógica del Sistema

Proceso general del funcionamiento de GameByte Store

Ventas Realizadas

Este código permite ingresar la cantidad de ventas realizadas en un día, registrar sus valores y mostrar un resumen con:

Descripción del Programa


  • Total de ventas realizadas
  • Promedio general
  • Venta más alta y más baja
  • Ventas que superan el promedio
El programa utiliza estructuras de control, arreglos y ciclos para procesar los datos.


Código Java


        import java.util.Scanner;

        public class GameByte {
            public static void main(String[] args) {
                Scanner sc = new Scanner(System.in);

                System.out.print("¿Cuántas ventas se realizaron hoy? ");
                int cantidadVentas = sc.nextInt();

                if (cantidadVentas == 0) {
                    System.out.println("No se realizaron ventas hoy ");
                    return;
                }

                double[] ventas = new double[cantidadVentas];

                for (int i = 0; i < cantidadVentas; i++) {
                    System.out.print("Ingrese el valor de la venta #" + (i + 1) + ": ");
                    ventas[i] = sc.nextDouble();
                }

                double total = 0;
                double maxVenta = ventas[0];
                double minVenta = ventas[0];

                for (double venta : ventas) {
                    total += venta;
                    if (venta > maxVenta) {
                        maxVenta = venta;
                    }
                    if (venta < minVenta) {
                        minVenta = venta;
                    }
                }

                double promedio = total / cantidadVentas;

                System.out.println("\n=== Resumen del Día ===");
                System.out.println(" Total de Ventas: $" + total);
                System.out.println(" Promedio de Ventas: $" + promedio);
                System.out.println(" Venta más Alta: $" + maxVenta);
                System.out.println(" Venta más Baja: $" + minVenta);

                System.out.println("\n Ventas mayores al promedio:");
                boolean hayMayores = false;
                for (double venta : ventas) {
                    if (venta > promedio) {
                        System.out.println("- $" + venta);
                        hayMayores = true;
                    }
                }

                if (!hayMayores) {
                    System.out.println("Ninguna venta superó el promedio.");
                }
            }
        }

Este proceso asegura que toda la información fluya correctamente, desde el usuario hasta la base de datos y el sistema pueda procesar las compras de forma ordenada.
